# CHANGELOG v2.3-alpha

## Added

- Instructor-mode backend helpers in `api/instructor.py`.
- Instructor presets endpoint.
- Instructor notes, bookmarks, and diagnosis-visibility state per session.
- Instructor report endpoint with JSON and Markdown output.
- Instructor panel in the web monitor UI.
- Documentation and regression tests for instructor mode.

## Scope

This version does not modify the physiology engine. It adds an instructor-facing orchestration layer above the v2.2 web training console.
