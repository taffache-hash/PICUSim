# PICUSim / Pediatric Critical Care Sim

PICUSim / Pediatric Critical Care Sim is a modular pediatric critical-care physiology simulation framework for education, scenario design, in-silico research prototyping, software testing, and hypothesis generation.

**Clinical status:** not for clinical use; not a medical device; not a validated patient-specific digital twin.

## Recommended OSF Structure

This OSF project is intended to complement the Zenodo software archive. Zenodo is the citable software source of truth; OSF organizes project context, validation artifacts, scenario-library documentation, metadata, and future supplementary material.

Suggested components:

1. **Software archive and citation**
   - Link to Zenodo DOI
   - Link to GitHub repository
   - Include `docs/PUBLIC_ARCHIVE_RELEASE_v6.0.md`

2. **Scenario library**
   - Describe the YAML scenario library
   - Point readers to `scenarios/` in GitHub/Zenodo

3. **Validation and regression outputs**
   - Include regression and validation summaries
   - Suggested files: `docs/REGRESSION_SWEEP_v5.3.md`, `outputs/regression_sweep_v5.3/regression_summary_v53.json`, `metadata/package_facts_v5.8.json`

4. **Documentation and metadata**
   - Include safety disclaimer, license, citation files, release notes, and archive manifest

5. **Manuscripts - deferred**
   - Manuscripts, paper figures, and final data/code availability statements remain deferred until after public identifiers and OSF structure are stable.

## Safety Notice

This software is for education, scenario design, in-silico research prototyping, software testing, and hypothesis generation. It must not be used for diagnosis, treatment, prescribing, triage, bedside monitoring, prognostication, device control, or patient-care decisions.
